# E-Commerce Sales Analysis 📊

## Project Overview

This project performs an end-to-end analysis of an e-commerce sales dataset to uncover key business insights such as revenue trends, top-selling products, and customer behavior.
The analysis includes data cleaning, exploratory data analysis (EDA), and visualization using Python.

---

## Objectives

* Analyze sales performance across different products and cities
* Identify top-performing product categories
* Discover monthly sales trends
* Understand customer purchasing patterns


## Technologies Used

* **Python**
* **Pandas**
* **NumPy**
* **Matplotlib**
* **Seaborn**
* **Jupyter Notebook**

## Key Insights

* Electronics category generates the highest revenue.
* Major sales come from metro cities.
* Sales show significant peaks during festive months.
* Customers aged 25–35 contribute the largest share of purchases.
